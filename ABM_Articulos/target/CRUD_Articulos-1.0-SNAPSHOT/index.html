Proyecto Artículos

Esta aplicación permite registrar un articulo,cualquiera que de los que se puede encontrar en un mercado de barrio.

La aplicación permite mediante un CRUD registrar los articulos en una base de datos

Los temas incluidos en este son

  • Herencia
  • Encapsulamiento
  • Abstracción
  • Polimorfismo
  • Interfaces
  • Patrón DAO
  • Base de datos
  • Java Server Pages(JSP)
